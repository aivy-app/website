プライバシーポリシー

最終更新日: 2026年5月10日
適用アプリ: AIvy（VRM デスクトップマスコット）

1. はじめに

本プライバシーポリシー（以下「本ポリシー」）は、AIvy（以下「本アプリ」）の運営者が、本アプリの利用に伴って収集する情報の取扱いについて定めるものです。本アプリをインストール・利用された時点で、本ポリシーに同意いただいたものとみなします。

本アプリは現在β版として提供されています。β版期間中は仕様および本ポリシーが予告なく変更される場合があります。

2. 運営者情報

運営主体AIvy 開発者（個人開発・日本国内）
連絡先aivy.support@gmail.com

※ 個人開発・β版段階のため、運営者の氏名・住所は非公開とさせていただきます。お問い合わせは上記メールアドレスまでお願いいたします。

3. 収集する情報

3.1 自動的に収集される情報

本アプリは、機能改善および不具合検出のため、以下の匿名情報を自動的に収集します。

項目内容送信先
機能利用統計設定タブの閲覧、機能切替（言語・プリセット・LLM/TTS エンジン選択など）Firebase Analytics
セッション情報アプリ起動・終了、セッション時間Firebase Analytics
環境情報OS 種別、アプリバージョン、表示言語、おおよその国（IP アドレスから自動判別）Firebase Analytics / Sentry
クラッシュ情報スタックトレース、エラー発生時の操作履歴（個人特定情報は自動でマスク）Sentry
デバイス識別子Firebase Instance ID（同一インストール内で一貫する匿名 ID。個人を特定しない）Firebase Analytics

3.2 本アプリが収集しない情報

以下は明示的に収集対象外であり、外部サーバへ一切送信されません。

  • ユーザーが入力したテキスト・コンパニオンとの会話内容
  • LLM / TTS の API キー、ローカルファイルパス、認証情報
  • VRM モデルファイル、3D アセット、音声データ
  • 氏名、メールアドレス、住所、電話番号などの個人特定情報
  • カメラ・マイクの入力、デスクトップ画面のスクリーンショット
  • クリップボードの内容

4. 情報の利用目的

  • 本アプリの機能改善・不具合修正
  • 利用統計の把握およびユーザー体験の最適化
  • クラッシュ・エラーの原因究明と再発防止

5. 第三者サービス

本アプリは以下の第三者サービスを利用しており、収集した情報の一部はこれらのサービスを通じて処理されます。

5.1 Google LLC（Firebase Analytics）

  • 用途: 匿名利用統計の集計
  • データ転送先: 米国（場合により他国も含む）
  • プライバシーポリシー: https://policies.google.com/privacy

5.2 Functional Software, Inc.（Sentry）

  • 用途: クラッシュレポートおよびエラー収集
  • データ転送先: 米国
  • プライバシーポリシー: https://sentry.io/privacy/

6. データの保管期間

サービス保管期間
Firebase Analytics最大 14 ヶ月（自動削除）
Sentry最大 90 日間（自動削除）

7. Cookie について

本アプリはデスクトップアプリケーションのため、Web ブラウザの Cookie は使用しません。ただし Firebase Instance ID（インストールごとに一意な匿名識別子）を Cookie に類する識別子として利用しています。当該 ID は本アプリのアンインストールにより消去されます。

8. オプトアウト方法

利用統計・クラッシュレポートの送信を停止したい場合は、いつでも以下の手順で無効化できます。

  1. 本アプリ右クリック → 「設定」を開く
  2. 「一般」タブを選択
  3. 「使用状況統計」セクションの「匿名の使用状況統計を送信する」のチェックを外す

オプトアウト後は Firebase Analytics・Sentry いずれの送信も即時停止します。既に送信済みのデータについては、第8項の保管期間が経過した時点で自動削除されます。

9. ユーザーの権利

EU・英国にお住まいの方は GDPR / UK GDPR に基づき、以下の権利を有します。

  • 収集された自身に関するデータへのアクセス権
  • 不正確なデータの訂正権
  • データ削除権（忘れられる権利）
  • データ処理の制限権
  • データポータビリティ権
  • 処理に対する異議申立権

ただし、本アプリが収集するのは匿名統計情報のみで、特定個人を識別できるデータは保持していません。データ削除等のご請求については aivy.support@gmail.com までご連絡ください。

10. お子様のプライバシー

本アプリは13歳未満のお子様による利用を想定していません。13歳未満の方の情報を意図的に収集することはありません。お子様が本アプリを利用していると判明した場合は、上記連絡先までご連絡ください。

11. 本ポリシーの変更

本ポリシーは予告なく変更される場合があります。重要な変更がある場合は、本アプリ内または本ページ上で通知します。最新のポリシーは本ページに常時掲載されます。

3. Information We Collect

3.1 Automatically Collected Information

The App automatically collects the following anonymous information for product improvement and error detection.

3.2 Information We Do NOT Collect

The following are explicitly out of scope and never sent to external servers:

  • Text input by the user / conversation content with the companion
  • LLM / TTS API keys, local file paths, credentials
  • VRM model files, 3D assets, voice data
  • Personally identifiable information (name, email, address, phone)
  • Camera / microphone input, desktop screenshots
  • Clipboard contents

4. Purpose of Use

  • Improving features and fixing bugs
  • Understanding usage statistics and optimizing user experience
  • Investigating and preventing crashes and errors

7. Cookies

As a desktop application, the App does not use web browser cookies. However, the Firebase Instance ID (a unique anonymous identifier per installation) functions as a cookie-equivalent identifier. This ID is removed when the App is uninstalled.

8. Opt-out

You can disable usage analytics and crash reports at any time:

  1. Right-click the App → open Settings
  2. Select the General tab
  3. Uncheck "Send anonymous usage analytics" in the Usage Analytics section

After opt-out, transmission to both Firebase Analytics and Sentry stops immediately. Previously sent data is auto-deleted after the retention period in Section 6.

The App only collects anonymous statistical information and does not retain data identifying specific individuals. For erasure or other requests, please contact aivy.support@gmail.com.
